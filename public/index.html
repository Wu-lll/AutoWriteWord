运行说明

按以下顺序使用即可：连接检测 -> 生成正文 -> 回看过程 -> 导出结果。

步骤 1：连接检测

填写 API 参数后，点击“测试连接”，确认接口可用再开始生成。

步骤 2：开始生成

填写故事设定并提交，系统会按阶段写作。你可实时查看当前状态和关键摘要。

步骤 3：复盘与导出

在“过程/提示词”中查看细节，在“正文”中切换阅读模式并导出 HTML、PDF、Markdown、TXT。
